## How to install?

```sh
$ curl -L https://raw.github.com/zenorocha/dotfiles/master/bootstrap.sh | sh
```

This will clone this repository and symlink the appropriate files in `~/.dotfiles` to your home directory.

## How to update?

Just go to `~/.dotfiles` and pull.

```sh
$ git pull
```
